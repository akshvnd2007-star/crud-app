# CRUD Application

A Student Management CRUD Application developed using Node.js, Express.js, EJS, CSS, and JSON.

## Features
- Create Student Records
- Read Student Records
- Update Student Records
- Delete Student Records
- Dark Mode
- Responsive UI
- Animated Cards and Effects

## Technologies Used
- Node.js
- Express.js
- EJS
- CSS3
- JSON
